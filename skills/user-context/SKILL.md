# User Context

1. `.agents/users/${user_name}.md`（`${user_name}` は `whoami` の出力）を読む。これが実行ユーザー個人のプロファイル。
   - **無ければ何もしない**（無理にヒアリングしない）。
2. front matter の `disabled_skills` に挙がったスキルは、ユーザーの指示なしで暗示的に呼び出さないこと。

## プロファイルの作成・更新（incremental）

最初に全部を聞き出す必要はない。**次のときだけ**、言われた分のセクションだけを作成・更新する（少しずつ育てる）:

- ユーザーが明示的にプロファイル作成を指示したとき。
- ユーザーがコミュニケーションスタイル / 非目標(non-goal) / 特定ツールの好み(skill preference など) を指定したとき。

書式・セクション・記法は `assets/user-profile-template.md` を参照（必要なときだけ読む）。更新したら変更内容を要約して次回からそれに従うようにファイルに記録したことをユーザーに伝える。
